# e-Commerce Curso React JS

Este es un trabajo final para el curso de React JS de CoderHouse, se trata de una e-commerce hecha en react que se conecta con una base de datos de firestore que cumple las funciones de backend, donde a traves de un carrito y un formulario, se pueden comprar productos desde un cat√°logo y luego enviar la orden hacia el backend conteniendo los datos de la persona que compra y los productos pedidos.
